## English

### 📚 What are Skills?

Skills are predefined instruction sets that help AI assistants automatically complete specific tasks.

### 📋 Available Skills

| Skill Name | Category | Description | Docs |
|-----------|----------|-------------|------|
| **Botdeck-Install** | Setup | Install and configure BotDeck MCP Server | [View](./botdeck-install/SKILL.md) |

### 🚀 How to Use

Reference skill docs in your AI assistant:
```
Please use Botdeck-Install skill to install MCP Server
```

### 🤝 Contributing

Welcome to submit new skills! Each skill should include:
- SKILL.md documentation
- Clear usage instructions
- Practical testing verification
