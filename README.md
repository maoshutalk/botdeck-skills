## English

### 📚 What are Skills?

Skills are predefined instruction sets for OpenClaw AI agents to automatically complete specific tasks.

### 📋 Available Skills

| Skill Name | Category | Description | Docs |
|-----------|----------|-------------|------|
| **Botdeck** | Setup | Install and configure BotDeck MCP Server + HTTP Bridge | [View](./botdeck-install/SKILL.md) |

### 🚀 How to Use

**For OpenClaw Users:**

1. Install this skill to your OpenClaw skills directory
2. Ask your OpenClaw agent:
```
Please use @botdeck-skills/botdeck-install/SKILL.md to install BotDeck
```

This will automatically:
- ✅ Install BotDeck MCP Server
- ✅ Install BotDeck MCP HTTP Bridge CLI
- ✅ Configure and test both components
- ✅ Enable all agents to use BotDeck tools

### 🤝 Contributing

Welcome to submit new OpenClaw skills! Each skill should include:
- SKILL.md documentation
- Clear usage instructions
- Practical testing verification
