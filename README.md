## English

### 📚 What are Skills?

Skills are predefined instruction sets for OpenClaw AI agents to automatically complete specific tasks.

### 📋 Available Skills

| Skill | What it does | Usage |
|-------|--------------|-------|
| **Botdeck** | Install BotDeck MCP Server + HTTP Bridge | Copy `@botdeck-skills/botdeck-install/SKILL.md` to OpenClaw |

### 🚀 How to Use

**Just paste this in OpenClaw:**
```
@botdeck-skills/botdeck-install/SKILL.md
```

Done! OpenClaw will install everything automatically.

### 🤝 Contributing

Welcome to submit new OpenClaw skills! Each skill should include:
- SKILL.md documentation
- Clear usage instructions
- Practical testing verification
